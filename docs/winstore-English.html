About Auto-set Lockscreen

View in Windows Store

Usage

This app automatically changes the lock screen with the photos or images.

How to add photos or images

With the following procedures, photos or images will be added.
  • Start this app, and tap [Add] button in [App Bar], and then select photos or images.
  • Copy photos or images to clipboard, and then start this app, and tap [Paste] button in [App Bar], and then tap [Add] button.
  • Select this app in [Charm]竊端Share] while using an other app, and then tap [Add] button. If an other app does not share photos or images, this app cannot be selected in [Charm]竊端Share].
This app does not automatically add photos or images.
The number of photos or images is up to 20 for the trial version, and up to 100 for the full version. When the total number exceeds this limitation, this app automatically removes photos or images which are already added.

How to remove photos or images

With the following procedures, photos or images will be removed.
  • Start this app, and select thumbnail images, tap [Remove] button in [App Bar].

How to set photos or images as the lock screen manually

With the following procedures, photos or images will be set as the lock screen manually.
  • Start this app, and select a thumbnail image, tap [Set as Lock Screen] button in [App Bar].

How to change the settings

Select [Charm]竊端Setting]竊端Auto-set settings], and then the setting popup will appear. The settings are as follows:
Change the lock screen when the app starts.
Specify whether this app changes the lock screen when this app starts. Specify whether this app asks you before changing the lock screen.
Change the lock screen periodically (only when AC power supply)
Specify whether this app changes the lock screen periodically. This changing occurs when powered by AC power supply.
Change the lock screen when a network change occurs.
Specify whether this app changes the lock screen when a network change occurs. This changing occurs when powered by either AC power supply or battery.
Minimum interval of changing
Until this time elapses after changing the lock screen, this app does not change the lock screen even if above three is triggered.

Privacy Policy

This app changes the lock screen with the photos or images. This app does not automatically add photos or images. This app does not use photos or images except for changing the lock screen. The photos or images added to this app are located in their original location or the local folder of this app. To remove photos or images, follow the procedure written in "How to remove photos or images".
